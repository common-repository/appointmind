== Description ==

Include your Appointmind online appointment scheduling calender in any article or in the sidebar. This plugin requires that you have purchased either a monthly subscription or the downloadable version of the software. This plugin does not include the appointmind scheduling software. You can get the subscription or the software at <a href="https://www.appointmind.com/?tracking=wordpress" target="_blank">Appointmind.com (English)</a> or at <a href="https://www.appointmind.de/?tracking=wordpress" target="_blank">Appointmind.de (Deutsch)</a>.

== Installation ==

1. Search for "Appointmind" under `Plugins > Add New` and click `Install Now`.
2. Activate the plugin on the `Plugins` page in WordPress.
3. Add your calendar URL in 'Settings' -> 'Appointmind Calendar'.
4. Go to Appearance -> Widgets and drag the Appointmind widget to the sidebar.
5. Add the placeholder [appointmind_calendar] to any post. Alternatively you can find short code for individual calendars and reasons in your Appointmind control panel under "Configuration / Schedules".
